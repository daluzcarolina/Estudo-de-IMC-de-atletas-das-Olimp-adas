# Estudo de IMC de Atletas das Olimpíadas

Análise exploratória do Índice de Massa Corporal (IMC) de atletas olímpicos, investigando padrões por modalidade esportiva, gênero e país de origem.

## Sobre o projeto

O objetivo deste estudo foi entender como o IMC varia entre diferentes perfis de atletas olímpicos, identificando tendências que diferenciam esportes de força, resistência e precisão.

## Perguntas investigadas

- Como o IMC se distribui entre as diferentes modalidades olímpicas?
- Existe diferença significativa de IMC entre atletas masculinos e femininos?
- Quais países apresentam os perfis físicos mais distintos?
- Esportes de força têm IMC consistentemente mais alto que esportes de resistência?

## Tecnologias utilizadas

- **Python** — linguagem principal
- **Pandas** — manipulação e limpeza de dados
- **NumPy** — cálculos estatísticos
- **Matplotlib / Seaborn** — visualizações
- **Jupyter Notebook** — ambiente de análise
- **Kaggle** — fonte do dataset

## Etapas da análise

1. Carregamento e inspeção do dataset
2. Limpeza de dados: tratamento de valores nulos e inconsistências
3. Análise estatística descritiva (média, mediana, desvio padrão)
4. Comparação de IMC por modalidade esportiva
5. Análise por gênero e por país
6. Visualizações comparativas

## Principais achados

- Atletas de esportes de força (halterofilia, luta, remo) apresentam IMC médio significativamente maior do que atletas de esportes de resistência (maratona, ciclismo)
- A distribuição de IMC varia consideravelmente entre continentes, refletindo diferenças em contexto esportivo e físico
- Algumas modalidades apresentam perfis físicos muito específicos, com baixa variância de IMC dentro do grupo

## Aplicações práticas

Os padrões identificados nesta análise têm usos concretos em diferentes contextos:

- **Seleção e treinamento esportivo:** comissões técnicas podem usar perfis de IMC por modalidade como referência para identificar atletas com potencial físico alinhado à demanda do esporte
- **Nutrição esportiva:** nutricionistas e preparadores físicos podem cruzar os dados de IMC com necessidades calóricas típicas de cada modalidade para personalizar planos alimentares
- **Medicina esportiva:** padrões de IMC muito fora da média por modalidade podem sinalizar risco de lesões ou problemas de saúde, auxiliando em triagens preventivas
- **Políticas públicas de esporte:** governos e federações podem usar os dados para entender quais perfis físicos seus países produzem e direcionar investimentos em modalidades com maior aderência
